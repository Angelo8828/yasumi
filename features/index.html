Features

Yasumi focuses primarily on a country's official holidays and non-working days. If time permits, other type of holidays will be added. Feel free to contribute to Yasumi to expand it's features and make it better!

Filters

Yasumi comes with filters that allow you to select only certain type of holidays. The following filters are currently available:

  • Official - Returns all holidays that are considered official
  • Observed - Returns all holidays that are considered to be observed
  • Bank - Returns all holidays that are considered bank holidays
  • Seasonal - Returns all holidays that are considered seasonal type of holidays
  • Other - Returns all holidays that do not fall into any of the above types
  • Between - Filters all holidays between a given start and a given end date
  • On - Filters all holidays that happen on a given date
